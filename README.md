# sbg-node-client
Seven Bridges NodeJS Client implementation

# Develpoment setup: 

    npm install -g gulp-cli tsd 
    
    # in root directory run
    npm install
    tsd install
    
    
    
create  file in `~/.sbg/configuration.json`


with content

    {
        "auth_token" : "xxxxxxx"
    }
    


### Running tests:

    gulp test
